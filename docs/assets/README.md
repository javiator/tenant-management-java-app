# Documentation Assets

This directory contains assets used in the Tenant Management System documentation and presentations.

## Directory Structure

```
docs/assets/
├── README.md                    # This file
└── screenshots/                 # Demo screenshots
    ├── README.md               # Screenshot documentation
    ├── dashboard_overview.png  # Main dashboard
    ├── properties_list.png     # Properties management
    ├── add_property_form.png   # Property creation form
    ├── tenants_list.png       # Tenants management
    ├── add_tenant_form.png     # Tenant creation form
    ├── transactions_list.png   # Transactions management
    ├── add_transaction_form.png # Transaction creation form
    ├── api_documentation.png  # API documentation
    └── api_endpoint_detail.png # API endpoint details
```

## Usage

These assets are referenced in:
- `Tenant_Management_System_Live_Demo_Presentation.md` - Main demo presentation
- Architecture documentation in the `docs/` directory

## File Formats

- **Screenshots**: PNG format for high-quality presentation
- **Documentation**: Markdown format for easy editing and version control

## Maintenance

When updating screenshots:
1. Capture new screenshots using the same naming convention
2. Update the README.md files with new descriptions
3. Update any references in the main documentation
